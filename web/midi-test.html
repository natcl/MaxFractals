Troubleshooting Tips

✔ Twister detected by system: lsusb shows ID 2580:0007
✔ Twister in ALSA: aconnect -i shows client 16: 'Midi Fighter Twister'
✗ Browser only sees "Midi Through Port-0" — permissions issue

If browser doesn't see the Twister:
1. Run groups in terminal — need to see audio in the list
2. If missing: sudo usermod -a -G audio $USER
3. Then completely restart the browser (close all windows, reopen)
4. If still missing: sudo apt install pipewire-audio or check /etc/group

Manual checks:
aconnect -i — list all MIDI input ports
lsusb — look for "DJ Tech Tools"
groups — check for audio group
cat /proc/asound/cards — sound card list
